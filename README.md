# Projeto Integrador 1 - PIJ110

Este é um repositório para a disciplina de PIJ110 da Univepsp

## Como executar
1. Instale o Python 3.12.12
2. Vá até a pasta do projeto e crie um ambiente virtual com `python -m venv .venv`
3. Ative o ambiente virtual com `.venv\Scripts\activate.ps1`
    1. Se estiver no Linux `source .ven/bin/activate`
4. Instale as bibliotecas requeridas com `pip install -r requirements.txt`. Isto pode demorar um pouco
5. Para executar algum dos testes basta usar `python teste_cliente.py`, por exemplo
6. Para executar o Streamlit use `streamlit run app.py`
